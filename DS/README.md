# 📊 Data Science – Rise 4.0

This folder contains all Data Science-related work for the **Rise 4.0 project**, including data preprocessing, exploratory data analysis (EDA).

---

## 🚀 Overview

The goal of this module is to understand historical data and extract meaningful insights to support decision-making.

Key objectives:
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Identify patterns and trends
